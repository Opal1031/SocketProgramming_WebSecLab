# STEP3: 백엔드

## 활동 내용
- Python Flask로 채팅 서버 구축
- Socket.io를 이용한 실시간 메시지 송수신 구현
- 프론트엔드(React)와의 연동

## 핵심 보안/학습 포인트
- HTTP 프로토콜 및 REST API 설계 이해
- 실시간 통신(Socket.io) 구조 및 패킷 흐름 분석
- 입력값 검증 및 데이터 위변조 방지
- 세션/쿠키의 역할과 인증 기본 개념

## 실습 예시
- `/api/messages` 엔드포인트로 메시지 송수신
- Socket.io를 통한 실시간 채팅방 구현
- 잘못된 입력값(예: 스크립트 삽입, 비정상 패킷) 처리 실습

## 실행 방법

1. Python 백엔드 서버 실행
	- 가상환경 활성화 후(또는 python, pip 명령이 바로 동작하면 생략)
	```
	python app.py
	```

2. 프론트엔드(React) 개발 서버 실행
	- 의존성 설치(최초 1회)
	```
	npm install
	```
	- 개발 서버 실행
	```
	npm run dev
	```

---

- 실습 후, API 보안 패치(입력값 검증, 인증 등) 필요성 인식
- 프론트엔드와 백엔드의 데이터 흐름 및 보안 연계 이해
